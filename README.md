# Diverse
A diversity App about news ,movies and other interesting things 
利用网上现有的接口，学习新的技术，写一个app.

### 项目架构
MVVM+Refrofit+Rxjava+kotlin

### 主要第三方依赖
- MVVM:com.github.goldze:MVVMHabit:3.0.6
- Okhttp
- Rxjava
- Retrofit
- Glide
- Gson
- Design
